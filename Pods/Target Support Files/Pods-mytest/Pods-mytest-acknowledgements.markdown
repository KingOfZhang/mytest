## Installation

### via CocoaPods
Install CocoaPods if you do not have it:-
````
$ [sudo] gem install cocoapods
$ pod setup
````
Create Podfile:-
````
$ edit Podfile
platform :ios, '5.0'
pod 'ZYQAssetPickerController',  '~> 1.0.0'
$ pod install
````
Use the Xcode workspace instead of the project from now on.
